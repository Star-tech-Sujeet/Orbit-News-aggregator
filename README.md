🌐 NewsOrbit — Circular News Aggregator

An interactive and visually stunning news aggregation web app that displays articles in a 3D circular orbit layout. Users can explore news dynamically with smooth animations, category filters, search, and bookmarking features.

🚀 Features
🌀 Orbit-Based UI – News cards rotate in a circular layout
🔍 Search Functionality – Find articles instantly
🏷️ Category Filters – Filter by Tech, Sports, Business, etc.
📖 Detailed Popup View – Click to read full article preview
🔖 Bookmark System – Save your favorite articles
🎯 Auto-Rotate Mode – Automatically rotates news cards
🎮 Interactive Controls
Mouse scroll / drag to rotate
Keyboard navigation (Arrow keys, Enter, ESC)
✨ Modern UI/UX
Glassmorphism design
Smooth animations
Responsive layout
📁 Project Structure
NewsOrbit/
